## 📌 Overview

This is the frontend of the Expense Tracker application built using **Next.js**.
It communicates with a Django REST API backend and provides a responsive financial dashboard.

---

## 🏗 Architecture

Frontend → Next.js (Vercel)
Backend → Django REST API (Render)
Database → PostgreSQL

🔗 Backend Repository:
https://expense-tracker-be-pomq.onrender.com/

---

## ✨ Features

* 🔐 JWT Authentication
* 📊 Expense Analytics Dashboard
* 📅 Monthly Filtering
* 🌙 Dark & Light Mode
* 📱 Mobile Responsive UI
* ⚡ API Integration using Axios
* 🧠 Global State Management (Redux Toolkit)

---

## 🛠 Tech Stack

* Next.js
* Redux Toolkit
* React Redux
* Axios
* Tailwind CSS

---

## ⚙️ Environment Variables

Create `.env.local`:

NEXT_PUBLIC_API_URL=https://expense-tracker-be-pomq.onrender.com/

## 🚀 Run Locally

git clone https://github.com/your-username/frontend-repo.git
cd frontend-repo
npm install
npm run dev
